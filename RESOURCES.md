# Anthropic 提示词工程资源

## Knowledge

- [Official: Prompting best practices — Claude API Docs](https://platform.claude.com/docs/en/build-with-claude/prompt-engineering/claude-prompting-best-practices)
  Anthropic 官方提示词工程最佳实践。三部分：模型特定指引 → 通用原则 → 迁移指南。核心参考源。
- [Anthropic Interactive Prompt Engineering Tutorial (GitHub)](https://github.com/anthropics/prompt-eng-interactive-tutorial)
  官方交互式教程，9 章 + 附录，含练习和答案。适合动手学习者。
- [PDF: 6 Techniques for Effective Prompt Engineering (Anthropic)](https://www-cdn.anthropic.com/62df988c101af71291b06843b63d39bbd600bed8.pdf)
  简明 PDF，覆盖 6 个核心技巧。
- [Effective Context Engineering for AI Agents — Anthropic](https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents)
  从提示词工程到上下文工程的演进。如何为 AI agent 准备最优上下文。
- [Building Effective AI Agents — Anthropic](https://www.anthropic.com/engineering/building-effective-agents)
  关于 agent 系统的工程实践，含工具提示词工程附录（ACI）。
- [Prompt Engineering Overview — Claude API Docs](https://platform.claude.com/docs/en/build-with-claude/prompt-engineering/overview)
  官方概览页面，含 prompt generator 介绍和定义成功标准的方法。

## Wisdom (Communities)

- [r/PromptEngineering (Reddit)](https://reddit.com/r/PromptEngineering)
  Reddit 提示词工程社区。讨论各模型的最佳实践、技巧和案例。
- [Anthropic Discord](https://discord.gg/anthropic)
  Anthropic 官方 Discord，有 prompt-engineering 频道，可直接与社区交流。

## Gaps

- 针对非 Claude 模型的跨模型提示词迁移策略 — 用户需要"不限于 Claude"的应用，目前资源以 Claude 为主，需要在课程中补充跨模型对比。
